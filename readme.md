## Pertanyaan Penelitian (Research Questions)
1. Parameter Estimation (Member B)
   Berapakah estimasi peluang (persentase) sebuah Pull Request (PR) di proyek React akan berhasil digabungkan (*merged*) dibandingkan dengan yang ditolak (*closed*)?
2. Inference & Hypothesis Testing (Member C & D)
   Apakah rata-rata waktu (dalam jam) yang dibutuhkan untuk menutup sebuah *issue* berlabel 'Bug' berbeda secara signifikan sebelum dan sesudah rilis versi utama (major release) terbaru?
3. Computational Analysis & Simulation (Member E)
   Berapakah peluang sebuah *issue* acak di repositori React akan tetap terbuka lebih lama dari 30 hari jika dihitung menggunakan simulasi Monte Carlo?
